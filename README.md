# Car Price Prediction using Machine Learning

## 📌 Project Overview

This project was developed as part of the CodeAlpha Data Science Internship.

The goal of this project is to predict the selling price of used cars based on various vehicle attributes such as age, mileage, fuel type, transmission type, and present market price.

Machine Learning helps estimate a fair resale value, which can benefit both buyers and sellers in the automobile market.

---

## 🎯 Problem Statement

Used car prices depend on several factors including:

* Vehicle Age
* Fuel Type
* Kilometers Driven
* Transmission Type
* Present Price

The objective is to build a machine learning model that predicts the selling price of a car using these features.

---

## 📊 Dataset Information

Features:

* Year
* Present Price
* Kms Driven
* Fuel Type
* Seller Type
* Transmission
* Owner

Target:

* Selling Price

## 📈 Machine Learning Model

Linear Regression

The model learns the relationship between vehicle features and selling price to generate accurate price predictions.

---

## 📊 Evaluation Metrics

The model is evaluated using:

* Mean Absolute Error (MAE)
* R² Score

Higher R² values indicate better prediction performance.
